# Compliance Types

This folder defines the data model for guideline checks and report outputs.

## Files
- `complianceIssueType.ts`: A single brand-color or prohibited-word violation.
- `complianceReportType.ts`: Per-product compliance summary with detected colors, blocked terms, and the final verdict.
- `complianceStateType.ts`: Feature-state shape for configured rules, issues, and stored reports.
