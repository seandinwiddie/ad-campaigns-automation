# Compliance Feature

This feature translates brand-guideline inputs into audit checks and per-product compliance reports.

## Composition
- `slice/`: Reducers, action re-exports, and selectors for colors, prohibited words, issues, and reports.
- `types/`: Issue, report, and feature-state contracts.
- `__tests__/`: Coverage for color matching, prohibited-word detection, and campaign-level compliance outcomes.
